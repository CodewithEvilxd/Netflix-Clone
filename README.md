### Netflix Clone V2 ⬇️
A Netflix clone, built with React, TypeScript, Firebase and Styled-Components. <br>
#### Live: https://netflix-miraya.vercel.app/  



### Features 🎉

- Mute & Unmute Trailer
- Authentication
- Get movies info 
- Save to watch list



### Dependencies 🛠️

* `Firebase`
* `React.js`
* `TypeScript`
* `Styled Components`
* `TMDB Movie API`
* `Build tool: Vite`
